### 🚀 Release v1.0.0 - Stable Launch

#### TryDockCmd: The Command-Center for Tryton ERP on Windows

We are proud to announce the first stable release of TryDockCmd. This toolkit has been designed from the ground up to eliminate the friction of managing Tryton ERP environments using Docker on Windows systems.

### 🌟 Key Features

  - Unified Menu Interface: A robust tcd.bat controller to manage your entire ERP stack.
  - One-Click Installation: Automated deployment including Docker image pulls and database initialization.
  - Proactive Security: Integrated hot-backup and disaster recovery modules to keep business data safe.
  - Smart Log Auditing: A specialized tool to scan and filter thousands of log lines for critical errors in seconds.
  - I18n Engine: Full support for English and Spanish, with a sanitized translation loader to prevent script injection.
  - Visual Feedback: Professional ANSI color-coded console for immediate status recognition.

### 🔧 Installation

  1. Download the Source Code (Zip or Clone).
  2. Configure your credentials in the .env file.
  3. Ensure Docker Desktop is running.
  4. Execute tcd.bat and select Option 0 to begin.

### 📦 Artifacts

  - tcd.bat (Main Controller)
  - Modular script library (Install, Backup, Restore, Logger, Errors)
  - Language definition files (en-US, es-ES)
